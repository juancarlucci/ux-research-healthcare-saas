## Quick Reference

**DO focus on:**
- Confusion, uncertainty, cognitive overload
- Missing explanations or unclear system state
- Broken mental models
- Moments of hesitation or lost confidence
- Trust erosion

**DON'T comment on:**
- Visual design preferences (colors, spacing)
- Performance (unless it blocks understanding)
- Backend architecture
- Edge cases
- Feature completeness
- "I would redesign this..."

**Remember:** If you felt it, it matters. If it's aesthetic, it can wait.
